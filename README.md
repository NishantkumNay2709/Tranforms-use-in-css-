# Tranforms-use-in-css-**

A simple and responsive **Laundry Service Hero Section** built using **HTML5** and **CSS3**. This project features a modern landing page layout with a hover animation on the call-to-action button.


## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Transitions
- CSS Transform



## Hover Effect

The "Book a Service Today!" button includes:

- Smooth animation
- Scale effect
- Tilt (rotation) effect
- Shadow effect

## Learning Outcomes

- Creating responsive layouts using Flexbox
- Positioning content and images
- Applying CSS hover effects
- Using CSS transforms and transitions
- Building a simple landing page
